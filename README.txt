You can see our maze design under assets/MazeDesignCheatSheet.PNG
The goal is to find spongebob squarepants and he is located in 3 places.
The maze is made using a text file:
each line represents a wall in the maze
every line is in this format:
row,cell,type

type is a number where it can be:
0 - east wall
1 - north wall
2 - goal point
